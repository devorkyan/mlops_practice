# Lab3: Контейнеризация микросервиса с помощью Docker

1. **Python-код для модели и микросервиса**  
   - Реализовано веб-приложение на Streamlit для классификации ирисов с использованием RandomForest
   - Модель обучается внутри контейнера при первом запуске, кэшируется
   - Файл: `app.py`

2. **Создание Dockerfile**  
   - Базовый образ `python:3.10-slim`
   - Установка зависимостей из `requirements.txt`
   - Копирование кода, открытие порта 8501, запуск Streamlit
   - Файл: `Dockerfile`

3. **Сборка Docker-образа**  
   - Команда: `docker build -t iris-classifier .`  
   - Образ успешно собран и протестирован локально

4. **Запуск Docker-контейнера и проверка**  
   - Команда: `docker run -p 8501:8501 iris-classifier`  
   - Приложение доступно по адресу `http://localhost:8501`, корректно обрабатывает запросы

5. **Использование docker-compose**  
   - Файл `docker-compose.yml` позволяет запустить сервис одной командой: `docker-compose up`

6. **Запуск приложения локально (без Docker)**

```bash
pip install -r requirements.txt
streamlit run app.py
```
